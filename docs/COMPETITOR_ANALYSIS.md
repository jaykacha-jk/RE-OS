# RE-OS Competitor Analysis

**Version:** 1.0  
**Market:** India real estate agency software

---

## 1. Competitive Landscape

| Competitor | Type | Strengths | Weaknesses | RE-OS differentiation |
|------------|------|-----------|------------|----------------------|
| **Sell.Do** | CRM + marketing | Established, automation | Heavy, expensive, complex | Simpler pipeline + AI calls |
| **LeadSquared** | Horizontal CRM | Flexible, integrations | Not real-estate native | Property inventory + SEO built-in |
| **Housing.com Agent tools** | Portal-linked | Lead volume | Portal lock-in | Agency-owned brand/site |
| **MagicBricks CRM** | Portal ecosystem | Brand | Limited customization | White-label tenant sites |
| **HelloLeads / niche CRMs** | Lightweight CRM | Cheap | No property SEO | Full OS + public listings |
| **Excel + WhatsApp** | Status quo | Free, familiar | No pipeline, no audit | Structured workflow + analytics |
